## Summary

Extend the inspector so memory curators can see likely duplicate clusters and possible update targets before memory quality issues spread.

## Goal

Turn similarity quality signals into useful inspection output that helps memory curators clean and govern the live corpus.

## Scope

In scope:

- inspector section for duplicate clusters
- inspector section for possible update targets
- links from diagnostics to memory ids and lineage
- thresholds or defaults suitable for memory curator review

Out of scope:

- editing memories directly from the inspector
- automatic cleanup actions
- changing retrieval ranking based on inspector output
