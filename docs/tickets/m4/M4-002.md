## Summary

Implement the full product scoring contract so retrieval ranking reflects trust, certainty, recency, and space weighting rather than similarity alone.

## Goal

Build the scoring engine used by retrieval with explicit support for confidence, trust, recency decay, and per-space weighting.

## Scope

In scope:

- cosine-based ranking contribution
- confidence weighting
- trust lookup from current policy
- episodic recency decay
- per-space default weights and request overrides

Out of scope:

- `FindCandidates` raw similarity mode
- audit or inspection ranking
- migration-time behavior
